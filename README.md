## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```shell
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

[http://localhost:4000/graphql](http://localhost:4000/graphql)

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
